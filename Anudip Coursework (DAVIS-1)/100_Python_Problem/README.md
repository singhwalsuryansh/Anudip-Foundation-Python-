## 🚀 How to Run

Each file is standalone and reads from `stdin`.

```bash
# Run any problem directly
python problem_01_swap_numbers.py

# Example with piped input
echo "5 10" | python problem_01_swap_numbers.py
```

- No external libraries required — pure Python only
